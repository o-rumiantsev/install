NAME
       tcp-server - test utility
SYNOPSIS
       tcp-server [ OPTION ] ...
DESCRIPTION
       simple  utility to start your own server which requires
       chat-client package.
OPTIONS
       --help,
             Output a usage message and exit.
       start [-l][ -p <port>],
             Start server on your computer.
             '-l' key requires logging to ./log/logs.txt
             '-p' key allows to set port.
       remove,
             Remove tcp-server utility from your computer.
       update,
             Update tcp-server to latest version.
       show-log,
             Output logs.
       clear-log,
             Delete all logs.
       -v, version
             Output version info.
